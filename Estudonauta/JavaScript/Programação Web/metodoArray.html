Métodos para Array

Métodos:

  • x.join("oque você quiser entre os itens") - para colocar 'algo a mais' entre cada item da matriz
  • x.pop() - para remover o ultimo item do array
  • x.push("oque voce quiser adicionar") - para adicionar itens ao final do array
  • x.shift() - para remover o primeiro item do array
  • x.unshift("oque voce quiser adicionar") - para adicionar itens no começo do array
  • x.splice('índice do item', 'quantos valores serão removidos', "os valores") - para adicionar ou remover valores em uma determinada posição
  • x.concat(y) - para concatenar matrizes
  • x.slice(índice da parte que será cortada) - para cortar um array
  • x.sort() - parada deixar os itens em ordem alfabética , caso queira em ordem 'desalfabética' basta colocar um x.reverse após o x.sort
    • x.sort(function(a,b){return a - b}) - para deixar números em ordem crescente
    • x.sort(function(a,b){return b - a}) - para deixar números em ordem decrescente
  • function maiorNum('array'){return Math.max.apply(null, 'array')} - para pegar o maior número de um array
    • function menorNum('array'){return Math.min.apply(null, 'array')} - para pegar o maior número de um array

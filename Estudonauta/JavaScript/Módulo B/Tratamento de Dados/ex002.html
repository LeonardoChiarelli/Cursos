Treinando comandos

  • Funções do '+'
    • (number + number) == adição
    • (string + string) == concatenação
  • Comando 'prompt'
    • Sempre retorna uma string
  • Convertendo string em number
    • Number.parseInt(n) - para números inteiros
    • Number.parseFloat(n)- para números reais
    • Number(n) - o próprio software se vira para saber se é float ou int
  • Convertendo number em string
    • String(n)
    • n.toString() - 'n' nesse caso indica valor

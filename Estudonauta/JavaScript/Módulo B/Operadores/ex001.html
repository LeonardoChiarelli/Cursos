Aritméticos

  • +
    • 5 + 2 = 7
  • -
    • 5 - 2 = 3
  • *
    • 5 * 2 = 10
  • /
    • 5 / 2 = 2.5
  • % - resto da divisão inteira
    • 5 % 2 = 1
  • ** - potenciação
    • 5 ** 2 = 25

Atribuição

  • let n = 3
    n = n + 7
    n = n - 5
    n = n * 4
    n = n ** 4
    n = n / 400
    n = n%6
    document.write(`${n}`)
  • Output: 4
  • Simplificação:
    • let x = 3
      x += 7
      x -= 5
      x *= 4
      x **= 4
      x /= 400
      x %= 6
      document.write(`${x}`)
    • Output: 4
    • Incremento:
      • let y = 5
        y++
        y--
        document.write(`${y}`)
      • Output: 5
      • y++ - a soma será feita depois
        ++y - a soma será feita antes (pre-incremento)
      • y-- - a subtração será feita depois
        --y - a subtração será feita antes (pre-decremento)

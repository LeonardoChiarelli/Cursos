Formulários com color, range e file

Nível de satisfação:

  • input:color:
    • Para adicionar uma cor inicial basta abrir o parâmetro "value" e colocar a cor em hexadecimal
  • input:range:
    • Para fazer com que a barra fique em uma determinada posição, basta abrir o parâmetro "value" e colocar um número de 0-100
    • Para mudar o "grau de nível" basta abrir as tag "min" e "max" e colocar os valores
      • Neste caso, se quiser mudar o "value" inicial, coloque os valores de min e max

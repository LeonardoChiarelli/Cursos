GRID LAYOUT

Anatomia do Grid

  1. grid CONTAINER - quem contém tudo
    1. grid ITEM - cada interseção entre coluna e linha
    2. grid TRACK - Quando temos vários itens na mesma linha e esses itens vão desde o inicio da grade até o final, tanto de maneira horizontal quanto de maneira vertical
      • row TRACK - Quando há uma track deitada
      • column TRACK - Quando há uma track em pé
    3. grid LINES - linhas entre os itens
      • Podem ser tanto verticais (delimitam as colunas), quanto horizontais (delimitam as linhas)
      • As tracks são SEMPRE numeradas e alinhadas da esquerda pra direita, no caso das verticais e de cima para baixo no caso das horizontais
        • A linhas mais à esquerda é numerada como 1
        • A linha mais à cima é numerada como 1
        • As linhas podem ter numeração negativa, que nesse caso começaria da direita pra esquerda como -1 (verticalmente) e de baixo para cima, também como -1 (horizontalmente)
    4. grid AREA - conjunto de itens tratados como agrupamento
    5. grid GAP - espaço entre os itens

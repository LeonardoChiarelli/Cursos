Criando caixas de texto e de senha

  • input:reset - Para apagar tudo do formulário
    • <input type="reset" value""> - Em "value" colocar o nome que você quer que apareça, no caso acima coloquei "limpar".
  • required - para tornar o preenchimento de um campo obrigatório.
    • <input type="password" name="" id="" required>
  • minlength e maxlength - comprimento mínimo e máximo
    • <input type="password" name="" id="" required minlength="3" maxlength="15">
    • Importante colocar um máximo de caracteres para as senhas, pois você terá que colocar esse máximo no banco de dados
  • size - para alterar o tamanho da caixa
    • <input type="password" name="" id="" required minlength="3" maxlength="15" size="15">
  • placeholder - "dica"
    • <input type="password" name="" id="" required minlength="3" maxlength="15" size="15" placeholder="senha">
  • autocomplete (dentro de input) - para ficar mais fácil do navegador sugerir senha, nome, sexo, entre outros parâmetros existentes.
    • a tag <form> deve estar com o "autocomplete="on"
    • <input type="password" name="" id="" required minlength="3" maxlength="15" size="15" placeholder="senha" autocomplete="current-password">

Ordem em uma mesma expressão:

  1. Aritméticos
  2. Relacionais
  3. Lógicos

Relacionais

  • < - menor que
    > - maior que
    <= - menor ou igual a
    >= - maior ou igual a
    == - igual a
    != - diferente de
    === - idêntico a (igualdade restrita)
    !== - desigual restrito

  • 5 < 4 = true
    4 > 5 = false
    6 <= 9 = true
    4 >= 8 = false
    12 == 12 = true
    14 != 14 = false
    5 === '5' = false
    5 !== '5' = true

    • Para o javascript 5 == '5' retorna 'true' pois ele testa os valores e não o tipo de variável

    Lógicos

    Ordem em uma mesma expressão:

    1. !
    2. &&
    3. ||
    • ! - negação
      && - conjunção (E)
      || - disjunção (OU)
  • !
    true -> false
    false -> true
    &&
    true && true -> true
    true && false -> false
    false && true -> false
    false && false -> false
    ||
    true || true -> true
    true || false -> true
    false || true -> true
    false || false -> false

    • Ternário

    • [teste] ? [true] : [false]
  • media >=6.0 ? "Aprovado" : "Reprovado"

  • let media = 5.5
    media>=6?"Aprovado":"Reprovado"
  • Output: Reprovado

  • let media = 5.5
    ++media
    media>=6.0?"Aprovado":"Reprovado"
  • Output: Aprovado
